# A01 - California Housing Boxplot 📊

This project loads the California Housing dataset using Python and creates a boxplot visualization of median income values from the dataset.

---

## 📂 Dataset

- California Housing dataset from `scikit-learn`

---

## 📁 Files

- `src/boxplot.py` → Python script used to load the dataset and generate the boxplot  
- `figs/boxplot.png` → Saved boxplot visualization output  
- `requirements.txt` → Required Python packages for the project  

---

## ▶️ How to Run Script

Install required packages:

```bash
pip install -r requirements.txt
```

Run the Python script:

```bash
python src/boxplot.py
```

---

## 📈 Expected Output

Running the script will generate and save the following figure:

```text
figs/boxplot.png
```

The boxplot visualizes the distribution of median income values (`MedInc`) from the California Housing dataset.
